# Vector Search with LLMs and Qdrant DB in Rust

## Prerequisites

- Rust
- Qdrant
- LLM

## Installation

```bash
# Clone the repository
$ git clone https://github.com/your-username/LLMQdrantVectorDB.git

# Change directory
$ cd LLMQdrantVectorDB

# Build the project
$ cargo build

# Run the project
$ cargo run
```

# Project: Talk with Database using RUST

## Description

This project is a simple example of how to use Rust to talk with a database.

## How to run

```bash

cargo run --quiet

```

## Helpful commands

```bash
ollama ls
```

## SQL Agent Commands

```bash
get xyz
```
